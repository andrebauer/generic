Module Generic_util_monad.IO

module IO: sig .. end

Similar to the Haskell IO Monad. The values are computations that may carry effects. The monadic bind is used to sequence the effects.

The constructor Generic_util_monad.IO.IO and synonym Generic_util_monad.IO.embed_io allow to embed an effectful computation in the monad. Generic_util_monad.IO.run_io runs it.
