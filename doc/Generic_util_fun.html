Function composition

The functions res_n compose a unary function to the result of a n-ary function, so that: 

res_n g f x_0 .. x_{n-1} == g (f x_0 ... x_{n-1})

Composition and Point-Free Operators


val (-<) : ('a -> 'b) -> ('c -> 'a) -> 'c -> 'b
(g -< f) x = g (f x)
val (>-) : ('a -> 'b) -> ('b -> 'c) -> 'a -> 'c
(f >- g) x = g (f x)
val (>>.) : ('a -> 'b) -> ('a -> 'c) -> 'a -> 'c
(f >>. g) x = f x ; g x
val (&&.) : ('a -> bool) -> ('a -> bool) -> 'a -> bool
(f &&. g) x = f x && g x

Isomorphisms


type ('a, 'b) iso = {
   fwd : 'a -> 'b;
   bck : 'b -> 'a;
}
An isomorphism is given by a bijection and its inverse, it must be true that: 
fwd (bck x) = bck (fwd x) = x
